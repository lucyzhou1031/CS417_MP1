# MP1

## Features Implemented:
A. Secrets: Hidden trophy and route to escape the room

B. Doors and Keys: there are 3 doors and corresponding keys to open them

C. Win Condition Scoreboard: On one side of wall in the room

E. Restart Option: Restart game with Left Primary Button (X)

G. Win Celebration: Trophy appear and congratulatory notes appear

H. Collectibles: keys and goldbars

J. Red Herrings: There are 5 key models in the scene but only 3 of these are real keys

K. Themed Setting: imported stone walls, wall-lamps and doors

L. Themed Props: imported keys, goldbar, and trophy models
